GatherBuddyReborn 是社区制作的最终幻想XIV原版 GatherBuddy 插件的分支版本。该工具旨在通过协助所有采集相关事务来增强您的游戏体验，现在通过 vnavmesh 支持自动路径导航。

## 功能特性

- **自动采集**: 自动寻路采集最多 10 组满堆叠（每组 999）的资源
- **资源队列**: 创建您想要的资源列表，GBR 将为每种资源采集最多 10 组！
- **完整的园艺/采矿自动化**: GBR 可以在世界中找到任何园艺/采矿物品并完全自动为您采集，除了初始设置外无需用户输入

**注意**: 完整自动化功能 *需要* vnavmesh 插件。有关 vnavmesh 的更多信息，请参阅本 README 的链接部分。
  
## 安装方法
- 在聊天窗口输入 `/xlsettings` 并进入打开窗口中的 Experimental 标签页
- **跳过 DevPlugins 部分，滚动到 Custom Plugin Repositories 部分**
- 将仓库链接复制粘贴到第一个空白文本输入框中
```
https://raw.githubusercontent.com/TonyCutHair/GatherBuddyRebornCN/main/repo.json
```
- 点击 + 按钮，确保之后新字段旁边的复选标记已设置
- **点击右下角的保存图标**

按照这些步骤，您应该能够在 Dalamud 插件安装器的 Available Plugins 标签页中看到所有包含的插件。
此时插件尚未安装，您只是让它们可用。现在您可以选择实际要安装的插件。
